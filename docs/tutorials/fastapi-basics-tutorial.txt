ðŸ“š Key Concepts to Know
-----------------------

- **FastAPI app instance:** The main object that handles requests.
- **Path Operations:** The routes or endpoints that define what happens at each URL.
- **Decorators:** Used to define HTTP methods such as @app.get(), @app.post(), etc.
- **Request & Response Models:** Define the data structure of inputs and outputs using Pydantic.
- **Automatic Docs:** Swagger UI (`/docs`) and ReDoc (`/redoc`) are generated automatically.
- **Uvicorn:** The ASGI server that runs the FastAPI application.
- **Dependency Injection:** A built-in system for reusing logic and managing components efficiently.
